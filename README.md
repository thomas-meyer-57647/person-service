# person-service
Zentrale Verwaltung von Personen-Stammdaten innerhalb eines Multi-Mandanten-Systems (je Company). Der Service bildet außerdem Teams und zeitliche Zugehörigkeiten ab (Eintritt/Austritt). Für HR-relevante Fälle wird „Mitarbeiter/Freelancer“ als Engagement (Beziehung) modelliert, nicht als feste Enum-Rolle in der Person.
